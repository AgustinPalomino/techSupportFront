Para Crear caso

Hay tres tipos de casos que usted puede crear:

Soporte para Software. This case type connects you to customer service for help with billing and account-related questions

Service limit increase requests are also available to all AWS customers. For information on the default service limits, see AWS Service

Limits Technical support cases connect you to technical support for help with service-related technical issues and, in some cases, third-party applications. If you have a Developer support plan, you can communicate using the web. If you have a Business or Enterprise support plan, you can also communicate by phone or live chat

Soporte para Software

Asistencia para windows, office, programas de oficina

Soporte Administrativo

Asistencia para software Administrativo, CRM, Aplicativos de negocio

Soporte de Hardware

Asistencia para incidencias con equipos de cómputo (portátiles, desktop, tablets) y periféricos (impresoras, drives, reguladores, etc.)
